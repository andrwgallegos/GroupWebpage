# GroupWebpage
# DXA Group Webpage

## Overview
This project is a front-end group webpage created for the **Group Webpage** assignment. The purpose of this activity is to practice basic web development skills using HTML, CSS, and JavaScript while building a page that represents our team identity.

Our team name is DXA.

The webpage includes:
- Individual team bios
- Photos/avatars for each member
- Unique computer science moodboards
- A team mission statement
- Team rules
- Custom CSS styling
- A team logo that represents our group

## Assignment Goals
This webpage was designed to meet the assignment requirements by including:

### 1. Modified Bios Page
Each team member has:
- A name
- A role/title
- A photo or avatar
- A short biography
- A moodboard with inspirations in computer science

### 2. Moodboards
Each bio includes a moodboard that reflects personal interests and inspirations such as:
- Favorite tools
- Useful resources
- Computer science concepts
- Creative influences

### 3. Team Mission Statement
The site includes a mission statement describing:
- Our team values
- Our goals
- Our strengths as computer science students

### 4. CSS Styling
Custom CSS was used to create:
- A cohesive team color palette
- Clear typography
- Distinct section styles
- Hover effects and transitions
- A responsive layout for different screen sizes

### 5. Team Logo
A custom DXA logo was included to replace the template logo and better represent our team identity.

## Files Included
- index.html – Main webpage structure and content
- styles.css – Page styling, layout, colors, and effects
- script.js – Interactive functionality for navigation and bio toggling
- img/ – Folder containing avatars and logo files

## Features
- Responsive team bio cards
- Expand/collapse bio sections
- Navigation buttons to switch between bios and mission section
- Unique moodboard sections for each member
- Styled mission statement and team rules
- Custom team branding

## How to Run the Project
1. Download or clone the repository.
2. Open the project folder.
3. Open index.html in a web browser.

No additional installation is required.
